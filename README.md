# CombatSlop YT
Browser extension that tags AI Slop videos on Youtube so you don't have to waste your time.

### How it works -
It relies on the wisdom of the crowd. Audience get to vote whether a video is slop or not, if a video has overwhelmingly amount of slop votes, it will be tagged. The tag can been seen on Home Feed and Search Results (by default).

### Planned Features -
- YT channel page support
- YT channel tags
- shorts support
- option to hide slop videos
- option to show only gems
